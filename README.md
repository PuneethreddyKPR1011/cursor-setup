## Issues Faced and Solutions

Issue 1: Git clone error

Solution:
Used a proper folder path and avoided invalid characters in the directory name

Issue 2: Claude extension login error
Error:
Anthropic.claude-code: Invalid argument

Solution:
Reinstalled the extension, restarted Cursor IDE, and logged in again

Issue 3: Git push not working

Solution:
Configured Git username and email using the following commands:
git config --global user.name "Your Name"
git config --global user.email "[your@email.com](mailto:your@email.com)"
