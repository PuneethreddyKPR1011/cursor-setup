## Issues Faced and Solutions

### Issue 1: Not a git repository

* Error: "fatal: not a git repository"
* Solution: Initialized git using `git init`

### Issue 2: Author identity unknown

* Error while committing
* Solution:

  * git config --global user.name "Your Name"
  * git config --global user.email "your email"

### Issue 3: Nothing to commit

* Cause: File not saved
* Solution: Saved files before running git commands

### Issue 4: Remote origin already exists

* Cause: Remote already configured
* Solution: Used existing remote without re-adding

## Key Learnings

* Strong hooks are critical for capturing attention
* Problem → solution storytelling is the most effective format
* Simple messaging performs better than complex explanations
* Deep understanding of ICP improves content performance
* Consistency is more important than perfection
* Value-first content builds trust and conversions
